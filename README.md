# RoboNone

シンプルな多機能サーバー管理Bot.

## ファイル構成

`bot.py` Botのメインファイル  
`run.py` 実行するファイル (secret tokenあり)  
`settings.json` 設定ファイル   
`run_heroku.py` herokuからの実行用  
`Procfile` heroku用設定ファイル  
`app.json` heroku用設定ファイル  

## コミット

prefix + 短い要約(日本語)

### prefix

`update:` ファイルの変更  
`add:` ファイルの追加  
`feat:` 機能の追加  
`refactor:` リファクタリング  
`remove:` ファイルの削除  
`fix:` バグの修正
